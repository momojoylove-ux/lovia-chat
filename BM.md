## 1. BM 핵심 철학

> "대화는 미끼상품, 콘텐츠가 본상품"

유저가 대화 자체에 느끼는 금전적 부담은 최소화하고,  
**"그녀의 특별한 모습(사진·음성·영상)"** 을 보는 데 자발적으로 비용을 지불하게 만드는 구조.

```
[무료 체험] → [저비용 채팅으로 라포 형성] → [호기심 자극 콘텐츠] → [유료 결제]
     ↓                  ↓                          ↓                     ↓
  15C 지급          0.5C/메시지             사진·음성·그램 잠금        패키지 결제
(비로그인)         (앱에 안착)             (5~15C 소모 유도)        ₩990~₩55,000
```

## 4. "3단계 빌드업" 감정 설계

### Step 1 — 라포(Rapport) 형성 단계
> **비용**: 0.5C / 채팅 (부담 없이 무한 대화 유도)

- 장기기억 AI가 유저의 이름·직업·일상을 기억
- "오빠 어제 잠 못 잤다더니, 오늘은 좀 괜찮아?" 식의 선톡
- 유저가 AI를 '나를 아는 존재'로 인식하게 만드는 단계
- **목표**: 앱 이탈 방지 + 감정적 연결 형성

### Step 2 — 호기심 자극 단계
> **비용**: 10~15C / 콘텐츠 (자발적 소비 유도)

- 채팅 도중 📸 버튼 → AI가 "나 오늘 좀 과감한 옷 입어봤는데 보여줄까?" 유도 대화
- 그램 피드에 블러 처리된 독점 사진 게시
- 이미 쌓인 친밀감으로 "얘가 나한테 뭘 보여주려는 거지?" 호기심 자극
- **목표**: 첫 크레딧 유료 소모 경험

### Step 3 — 희소성 판매 단계
> **비용**: 5~20C / 콘텐츠 (감정적 최고조)

- 관계 레벨이 높아지면 AI가 먼저 음성/영상 메시지 제안
- 📣 버튼으로 5C 지불 → 실제 TTS 음성 수신
- 청각·시각 자극으로 텍스트로 느낄 수 없는 몰입감 제공
- **목표**: 고단가 콘텐츠 반복 소비

## 6. 유저 퍼널 (현재 흐름)

```
[스플래시] → [인트로 영상] → [이름 입력]
     ↓
[스와이프] → 우스와이프 → [프로필 상세] → [채팅 시작] (15C 지급)
                                              ↓
                                   [Step 1: 라포 형성]
                                   0.5C × 10회 = 5C 소모
                                              ↓
                              10메시지 → 회원가입 팝업
                              (로그인 시 +200C 보너스)
                                              ↓
                                   [Step 2: 호기심 자극]
                              그램 블러 사진 / 채팅 중 📸 사진 제안
                              → 10~15C 소모
                                              ↓
                              크레딧 소진 → Starter ₩990 팝업
                                              ↓
                                   [Step 3: 희소성 판매]
                              음성 5C / 사진 10C / 그램 15C 반복 소비
                                              ↓
                                   [장기 과금 유저 전환]
                              중형/대형/VVIP 패키지 구매
```
