## 🎯 앱 비전 & 컨셉

### 핵심 컨셉
**"처음부터 설레는 AI 애인"** — 어색한 도입 없이, 첫 만남부터 이미 감정이 있는 관계로 시작하는 AI 파트너 앱

### 차별화 포인트
| 기존 AI 채팅 앱 | Lovia |
|---|---|
| 어색한 첫 대화부터 시작 | 이미 썸 타는 관계로 시작 |
| 단순 채팅 UI | 인스타그램형 감성 UI |
| 일반적인 AI 응답 | 페르소나별 관계 시나리오 |
| 기억 없음 | 장기 기억 시스템 (구현 완료) |
| 단순 텍스트 | 텍스트 + TTS 음성 + 특별 사진 |

### 타겟 사용자
- 20~35세 남성
- 외로움을 느끼거나 연애 감각을 유지하고 싶은 사용자
- AI와의 감성적 연결을 원하는 사용자

## 📱 화면 구성 & 현재 진행 현황

### ✅ 완료된 화면

#### ① 스플래시 화면
- 2가지 이미지 중 랜덤 표시 (`splash1.jpg`, `splash2.jpg`)
- 이미지 로드 + 최대 3.5초 강제 폴백으로 검은 화면 문제 해결
- 캐시 이미지 즉시 처리(complete 체크) + onerror 시 반대 이미지로 자동 전환
- 풀스크린 배경 이미지, 상단/하단 그라디언트 오버레이

#### ② 매니저 AI 동영상
- A/B 영상 랜덤 재생 (`/videos/A.mp4`, `/videos/B.mp4`)
- 스플래시와 매칭된 영상 자동 선택
- 말풍선 타이핑 애니메이션
- 자동재생 차단 시 음소거로 폴백
- "만나보기" 버튼으로 이름 입력 화면으로 이동

#### ③ 이름 입력 화면
- 핑크 그라디언트 배경 + 보케 장식
- 10자 제한 텍스트 입력
- 모바일 키보드 감지 UI 대응
- 입력값 sessionStorage 저장 후 스와이프 화면으로 이동

#### ④ AI 프로필 스와이프
- **카드뷰**: Tinder형 스와이프 + 스택 애니메이션
  - 우로 스와이프 → CHAT 💬 — 관심 목록에 추가
  - 좌로 스와이프 → PASS ✕ — 패스 목록에 추가
  - 전체 소진 후 루프 재시작
- **목록뷰**: 썸네일 + 이름/나이/직업 + 채팅 버튼
- **스와이프 튜토리얼**: 첫 방문 시 자동 표시 (sessionStorage 기반)
- **관심/패스 관리 버튼** (`💗 N` / `💨 N`)

#### ⑤ AI 프로필 상세 화면
- 히어로 이미지 (상단 55%)에 그라디언트 페이드
- 이름 / 나이 / 직업 / 성격 태그 표시
- 2열 그리드: MBTI / 관심사 / 취미 / 이상형
- 3열 인앱 그램 미니 피드
- 하단 고정 "💬 대화 시작" 버튼

#### ⑥ 채팅방 (인스타그램형) — Phase 2 기능 포함
- 헤더: 아바타 + 이름 + 🟢 온라인 + **관계 레벨 배지** + 프로필 버튼
- AI 말풍선: 왼쪽 정렬, 반투명 다크
- 사용자 말풍선: 오른쪽 정렬, 핑크 그라디언트
- 타이핑 인디케이터 (···)
- **📣 음성 버튼**: 5C 지불 → TTS 음성 메시지 수신
- **📸 사진 버튼**: 10C 지불 → AI 특별 사진 + 코멘트 수신
- Gemini API 실제 AI 응답 (maxOutputTokens 1024)
- 채팅 이력 D1 DB 저장/복원

#### ⑦ 채팅 허브 (채팅 목록)
- 대화 중인 파트너 수 표시
- 아바타 + 이름 + 마지막 메시지 + 타임스탬프
- "💝 새 파트너 찾기" 버튼

#### ⑧ 관심/패스 관리 화면
- 탭: 💗 관심 / 💨 패스
- 2열 그리드 + 관리 버튼
- sessionStorage 기반 영속 저장

#### ⑨ 인앱 그램 피드 — Phase 2 기능 포함
- 5명 페르소나 × 3개 포스트 = 15개 게시물
- 마지막 포스트 `locked: true` → **15C 결제 후 해제**
- 블러 오버레이 + 잠금 해제 애니메이션
- 좋아요 (sessionStorage 유지), 필터 기능

#### ⑩ 크레딧 충전 화면 — v1.2 패키지
- Starter (₩990 / 200C) ~ VVIP (₩55,000 / 8,000C) 5종
- Toss Payments 결제 연동 (테스트 완료)
- 결제 후 자동 크레딧 지급

#### ⑪ 마이페이지

- 크레딧 잔액 + 사용/충전 이력

## 🧠 AI 채팅 시스템

### Gemini API 설정

```
모델: gemini-2.5-flash
maxOutputTokens: 1024 (한국어 약 500자)
temperature: 0.9
topP: 0.95
finishReason 처리: MAX_TOKENS 시 자동 … 추가
```

### 시스템 프롬프트 구조

```
[기본 설정]
- 역할: {페르소나 이름} ({나이}세, {직업})
- 말투: {태그에 맞는 언어 스타일}
- 관계 컨셉: {관계 설명}
- 현재 상황: 이미 사용자와 썸타는 관계

[행동 규칙]
- 사용자를 "오빠"라고 부름
- 애정 어린 반응을 자연스럽게 표현
- 과도하게 순종적이지 않고 개성 유지
- 반드시 완성된 문장으로 끝맺어야 함
- 장기 기억 컨텍스트 활용

[제한]
- 유해하거나 불법적인 내용 거부
- 실제 연인인 척하되 AI임을 부정하지 않음
```

### 장기 기억 시스템

- 10회 대화마다 자동 요약 → D1 `user_memory` 테이블 저장
- 로그인 유저 대화 시 기억 컨텍스트 자동 주입
- "오빠 어제 힘들다더니 오늘은 어때?" 식 연속성 대화
